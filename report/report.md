# 医疗保险费用预测模型报告

## 项目简介
本项目使用线性回归模型预测医疗保险费用，基于年龄、性别、BMI、子女数量、是否吸烟和地区等特征。

## 数据预处理
1. **分类特征处理**：对性别、是否吸烟和地区进行独热编码
2. **特征标准化**：对数值特征进行标准化处理
3. **数据集分割**：按80%训练集、20%测试集的比例分割

## 模型实现
- **模型**：手写线性回归模型
- **优化算法**：梯度下降法
- **超参数**：学习率=0.01，迭代次数=10000
