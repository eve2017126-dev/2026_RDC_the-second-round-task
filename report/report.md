# 医疗保险费用预测模型报告

## 项目概述
本项目基于线性回归算法，构建了一个医疗保险费用预测模型。通过分析个人基本信息（年龄、性别、BMI、子女数量、吸烟状况、居住地区），模型能够准确预测个人的医疗费用支出。

## 数据探索与预处理

### 数据集基本信息
- **数据来源**：insurance.csv
- **样本数量**：1,338条记录
- **特征维度**：6个原始特征
- **目标变量**：医疗费用（charges）

### 数据预处理流程
1. **分类特征编码**：对性别（sex）、吸烟状况（smoker）、地区（region）进行独热编码
2. **数值特征标准化**：对年龄、BMI、子女数量进行Z-score标准化
3. **数据集分割**：80%训练集（1,070条），20%测试集（268条）
4. **偏置项添加**：为线性回归模型添加截距项
